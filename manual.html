Traffic Impact Assessment User Manual

Use this guide to run the calculator, load traffic datasets, build report outputs, and troubleshoot common issues.

Open Main View Open Formula View
Formula View keeps formulas and calculation steps visible throughout the report workflow.

1. Getting Started

  1. Open index.html for the standard calculator or index_formulas.html for formula trace mode.
  2. Sign in through the login prompt.
  3. Wait for traffic datasets to load. The top badge shows Sites Loaded.
  4. Search and select a site from the map or site search field.

2. Core Workflow

  1. Confirm site metadata: road name, directions, and counter details.
  2. Set analysis inputs: base year, target year, growth rate, lane settings, and assumptions.
  3. Review traffic split inputs for each direction (D1 and D2).
  4. Run calculations to generate hourly volumes, VCR, queue lengths, and peak period outputs.
  5. Check map markers, chart trends, and table values for consistency.

3. Report and Export

  • Switch to report mode to generate printable output.
  • Use print preview before exporting PDF.
  • Use share options where needed for report distribution.
  • In Formula View, formulas and step traces remain visible in report sections.

4. Data Sources

The app loads datasets from raw GitHub sources and falls back to local files if a source is unavailable.

5. Troubleshooting

  • No sites loaded: hard refresh the page and ensure internet access to raw GitHub files.
  • Map not visible: wait for library load and verify no browser extensions are blocking scripts.
  • Unexpected values: verify direction split, growth rate, heavy vehicle percentage, and year inputs.
  • Formula details missing: open index_formulas.html and regenerate the report.

7. FAQ

Q: Why does the app show 0 sites loaded?

A: Reload with Ctrl+F5. If still zero, check internet access and confirm raw GitHub dataset links are reachable.

Q: Which page should I use for calculations with visible formulas?

A: Use index_formulas.html. It enforces formula/step visibility in the output workflow.

Q: Can I still use the tool if one online dataset fails?

A: Yes. The loader attempts raw GitHub data first, then falls back to local dataset files in this folder.

Q: How do I create a final report PDF?

A: Enter report mode, review the sections, run print preview, then choose Print / Save PDF.
